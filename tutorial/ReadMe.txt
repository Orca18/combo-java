
 Please read the Reference Manual and the Programmers Manual before running the
 sample programs.

 1. Compiling a sample program:

	[Windows]
	javac -cp ..\PrFps.jar;. sample.java

	[Linux]
	javac -cp ../PrFps.jar:. sample.java

 2. Running the sample program:

	[Windows]
	java -cp ..\PrFps.jar;..\jna-3.4.1.jar;. sample

	[Linux]
	java -cp ../PrFps.jar:../jna-3.4.1.jar:. sample
